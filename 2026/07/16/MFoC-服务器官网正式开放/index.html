MFoC 服务器官网正式上线

🎉 MFoC 服务器官网正式上线！

MFoC 服务器官网于今日正式开放！🎊

📦 核心模组与玩法方向

⚙️ 机械动力（Create）

包含本体及 Aeronautics（飞艇）、Big Cannons（大炮）、New Age + Power Grid（电力）、Nuclear（核电）、Additional Logistics（物流无人机） 等全套附属，工业线内容非常深。

🍳 农夫乐事 + 森罗物语

农夫乐事提供完整的农耕烹饪体系，森罗物语在此基础上追加了中式炒菜、酿酒系统、酒馆经营等玩法。包含完整的葡萄种植、果汁压榨、发酵陈酿流程，品质越高的酒增益越强。

👘 车万女仆

可召唤并驯服女仆随从，在日常建造和探索中提供各类辅助。本服给这个模组接入了 DeepSeek V4 Pro 模型，与女仆对话是自然的，不是预设词条的复读机。

🐉 龙之生存

玩家开局选择一种龙族血统（火、冰、森林、海洋等），从幼龙开始成长，体型和能力随经验逐步提升。成年后解锁真实飞行和元素吐息攻击，支持切换回人形态方便做精细操作。

⚔️ 冒险与探索

  • Iron’s Spells n Spellbooks：100+ 法术
  • L_Ender’s Cataclysm、Eternal Starlight、Qliphoth Awakening、Beyond Dimensions 等模组提供新维度、新 Boss
  • 配合 Terralith + Tectonic 重做的地形，以及 YUNG 全家桶 重制的结构，探索体验与原版完全不同

🛠️ 辅助与优化

FTB Quests（引导）、FTB Ultimine（连锁挖矿）、FTB Teams（组队领地）、Lightman’s Currency（经济系统）、Jade / JEI / Xaero 地图 等工具齐全。优化方面 C2ME、ModernFix、Ferrite Core、Sodium 全部拉满。

🎯 服务器特色

🧩 内容丰富，但不堆砌

270+ 模组，每条玩法线都有深度。你可以钻研机械、下地种田、开酒馆、招募女仆、作为一条龙在天空翱翔，也可以出门打 Boss、探索新维度。几条路线互不冲突，可以同时进行。

🚀 性能优化到位

Create 的机械结构一旦做大了，对服务器压力不小。MFoC 在优化上下了不少功夫，不管是大面积传送带、多区块动态结构，还是几艘飞艇同时飞行，整体都跑得比较顺。

🌐 网络稳定

国内三网连接正常，电信、联通、移动各自走各自的线路，延迟体感差别不大。

📋 服务器规则

玩法倾向

鼓励生电自动化、大型建筑、探索与合作。不限制红石机械数量，但请合理设计，避免无效的卡服级运算（如无意义的高频脉冲）。

具体规则

  • 严禁恶意破坏他人建筑和设施
  • 严禁偷窃他人箱子和机器中的物品
  • 严禁使用作弊类模组或外挂（包括但不限于透视、飞行、加速等）
  • 严禁恶意刷屏、辱骂、引战
  • 大型机器建议做开关控制，不用时关闭减少服务端负载
  • 探索新区域时请回收或销毁多余的方块和实体，避免垃圾堆积

保护机制

机制 说明
方块记录回溯 任何破坏行为都有记录可查
领地保护（FTB Teams） 可设置领地权限
死亡不掉落（Corpse） 尸体保留物品
物品黑名单 部分道具被禁用，具体见群公告

🏛️ 关于 MFoC

这个服务器不换周目。

你建的工厂、铺的铁路、种下的田、招募的女仆，不会某天登录发现全没了。我们觉得 Minecraft 的乐趣有一部分就来自于时间的沉淀——建筑越老越有味道，机器越跑越有感情。

社区氛围比较随和。 没有勾心斗角的事，大家各玩各的，遇到问题喊一声基本都有人搭把手。偶尔有人搞点大型工程，群里会热闹一阵，平时就是安安静静各造各的。

模组不是定死的。 如果你觉得包里的模组不够玩，或者有什么想加但没加上的，直接跟服主说就行。只要模组本身合适（不破坏平衡、不严重卡服、没有恶性 Bug），都会进入审核流程，通过了就加。

活动方面，我们偶尔会搞一些轻松向的，比如划船比赛、鞘翅穿环、TNT 跑酷，最近也在琢磨一些组队合作的小游戏。不是每周必打卡的强制活动，想来的来，不想来的自己玩也行。

MFoC 的模组包内容深度足够玩上几个月。无论你喜欢钻研机械、种田酿酒、招募女仆，还是变成龙到处飞，这里应该都待得住。
